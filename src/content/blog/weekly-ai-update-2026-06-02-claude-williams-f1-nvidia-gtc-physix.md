## ①【最重要】Claude × Williams F1：ピットウォールにAIが入った

**出典**: The Next Web「AI is now an F1 sponsor, a strategist, and a tech director」

AnthropicのClaude AIが**Williams Racing F1チームの運営・レース戦略スタックに正式採用**された。Claudeはピットウォールでのリアルタイムデータ解析、複数の戦略オプション評価、コミュニケーション補助に活用される。同時にGoogleのGeminiがMcLaren F1の分析プラットフォームにも統合されており、GPT系モデルに偏りがちだったF1のAI導入が急速に多様化している。

注目すべきスペックデータ：**2026年のF1マシンは1台あたり300〜600個のオンボードセンサーを搭載し、1秒あたり100万データポイント以上を送信する**。人間のエンジニアがリアルタイムに処理できる量を遥かに超えており、AIによる即時フィルタリング・パターン抽出なしにはデータが活かせない。

さらに**FIA自体がAIを2026年の技術規則施行に投入**しており、グラウンドエフェクト・フレキシブルフロアなどの判定にアルゴリズムが使われる。審判まで自動化される時代に入った。

**MBDへの示唆**: テレメトリストリームにAIエージェントをつなぐアーキテクチャはレースに限らない。実車テスト・ダイナモ計測にも即応用可能だ。

```bash
# Claude CLIでCSVテレメトリを即解析（30秒で試せる）
# 前提: npm install -g @anthropic-ai/claude-code

claude "このCSVはF1テレメトリデータです。ラップごとの
ブレーキ温度最大値・最小値・平均値を表にまとめて、
異常値（平均±2σ超）があれば指摘してください" \
  --file telemetry_data.csv
```

## ②NVIDIA GTC 2026：「NeMoCLAW」でエージェント時代が本格化

**出典**: NVIDIA Newsroom（2026年6月1日）

NVIDIA GTC 2026（サンノゼ）でJensen Huang CEOが「**アシスタント型AIからエージェント型AIへ**の転換が起きた」と宣言。エンジニアリング分野に直結する発表が3つ：

| 発表 | 概要 | MBD関連性 |
|------|------|-----------|
| **NVIDIA OpenShell™** | ポリシーベースのセキュリティ・プライバシーガードレール付きエージェントランタイム（OSS） | 社内秘密情報を扱うMBDエージェントのセキュア化に使える |
| **NeMoCLAW** | マルチエージェント調整ツール。Claude Code・OpenCLAWと連携 | Simulinkモデル生成→検証→CI/CDを全自動化する基盤 |
| **物理AI Factory** | NVIDIA DoMINO + Digital Humanを組み合わせた製造ライン向けAIワークフロー | CFDサロゲート（DoMINO NIM）と設計エージェントの統合基盤 |

Jensen Huang曰く「**Claude CodeとOpenCLAWがエージェントの変曲点を引き起こした**」——Claude Codeがプログラミングエージェントの事実上の標準として業界に認知されつつある証拠だ。

NeMoCLAWとDoMINO NIMを組み合わせると「設計変数入力→CFD実行→結果解析→次設計案提案」のループがAIに全自動委任できる可能性が出てきた。

## ③PhysiX：4.5Bパラメータの「物理版ChatGPT」がarXivに登場

**出典**: arXiv:2506.17774（UCLA Tung Nguyen ら、2026年6月）

UCLAのTung Nguyen氏らが**PhysiX**を発表した。物理シミュレーション用の初の大規模基盤モデルで、4.5Bパラメータ・自己回帰Transformerが流体・熱・衝撃波・多相流を「次トークン予測」で生成する。

- The Wellベンチマーク全ドメインで既存基盤モデル＆特化型を上回る
- 自然動画からの転移学習で物理データ希少性問題を突破
- ゼロショットで未見の物理系を近似可能

これと並んでGPhyT（1.8TB学習、特化型比最大29倍精度）も注目度が高まっている。

一方、PhysicsX社（別会社・英国スタートアップ）は独自データセット（PXNetCar、2万件超のCFDシミュレーション）で学習したモデルを使い、欧州大手自動車メーカーとの共同で**既存最適設計より空力性能7%向上・質量10%削減**を達成。研究論文→実用化の速度が驚異的に速くなっている。
