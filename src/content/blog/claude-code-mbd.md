## はじめに

レース車両のモデルベース開発（MBD）は、Simulinkモデルの構築・検証・自動コード生成まで複雑なワークフローを持ちます。これまでエンジニアが手作業で行っていた多くのタスクが、**Claude Code**を使うことで大幅に効率化できます。

Claude Code は Anthropic が提供するターミナルベースのAIコーディングエージェントです。コードの読み書きだけでなく、ファイル操作・テスト実行・ドキュメント生成を**自律的**に行える点が特徴です。

---

## Claude Code がMBD開発で役立つ場面

### 1. MATLABスクリプトの自動生成・リファクタリング

走行データの前処理スクリプトや、Simulinkモデルへのパラメータ読み込み処理は定型的なコードが多く、Claude Code に任せることができます。

**活用例：**
- ログファイル（.mat / .csv）を読み込み、チャンネル別に整形するスクリプト生成
- 複数テストセッションのデータを一括処理するバッチスクリプト作成
- レガシーなMATLAB関数をクラスベースに書き換えるリファクタリング

```
# ターミナルでの使い方例
claude "走行データ（CSV形式）を読み込んで、
        速度・操舵角・Gセンサーをチャンネルごとに
        分けてmatファイルに保存するMATLABスクリプトを作って"
```

Claude Code は指示を受け取った後、ファイル構造を自分で確認しながらスクリプトを作成・修正します。

---

### 2. Simulinkモデルのドキュメント自動生成

Simulinkモデルのブロック構成説明やインターフェース仕様書の作成は、手動では時間がかかります。Claude Code を使えば、既存スクリプトや設計仕様をもとに**マークダウン・Word形式の仕様書を自動生成**できます。

**活用例：**
- モデル内のブロックリストから機能説明を生成
- シグナルフローの説明文を自動作成
- テストベクタ仕様書のテンプレート生成

---

### 3. テスト自動化とCI/CDパイプライン構築

MIL（Model-in-the-Loop）テストの実行スクリプトや、テスト結果を自動集計してレポートするPythonスクリプトの作成にも活用できます。

**活用例：**
- MATLAB Unit Test フレームワーク用のテストクラス生成
- テスト結果のHTMLレポート自動生成
- GitHub Actionsと連携したMATLABテスト自動実行スクリプト

---

### 4. 走行データ解析ツールの開発

Python + pandas/matplotlibを使った走行データ可視化ツールをClaude Codeに作らせることができます。

**活用例：**
- ラップタイム比較グラフ生成スクリプト
- タイヤ温度・摩耗データのトレンド解析ツール
- コーナリングデータのオーバーレイプロット

## 実際の使い方のコツ

### コンテキストを丁寧に与える

Claude Code は会話内で与えた情報をもとに動きます。以下のようにプロジェクト構造や前提条件を最初に説明しておくと精度が上がります。

```
"このプロジェクトはMATLAB R2024bを使用。
走行データはdata/フォルダにCSV形式で格納されている。
チャンネル名はヘッダー行に記載。単位はSI単位系。"
```

### CLAUDE.md を使ってプロジェクト設定を記録

プロジェクトルートに `CLAUDE.md` というファイルを置いておくと、Claude Code が毎回そのファイルを読んでプロジェクトの設定・規約を把握します。

```markdown
# CLAUDE.md
## プロジェクト概要
レース車両MBD開発プロジェクト。MATLAB R2024b / Simulink使用。

## コーディング規約
- 変数名はキャメルケース
- 関数の入出力は必ず型コメントを付ける
- 単位はSI単位系を使用
```

---

## 注意点と限界

- Simulinkモデル（.slx）ファイルをClaude Codeが直接編集するのは困難。MATLABスクリプト経由でモデルを操作する方法が現実的。
- MathWorksの最新MATLAB機能については学習データに含まれていない場合があるため、公式ドキュメントと合わせて確認する。
- 生成されたコードは必ず動作確認とレビューを行うこと。

---

## まとめ

Claude Code はMBD開発の「繰り返し作業」を大幅に削減できるツールです。モデル本体の設計はエンジニアが担い、スクリプト・ドキュメント・テストといった周辺作業をClaude Codeに任せる分担が効果的です。

まずは小さなスクリプト生成タスクから試してみることをおすすめします。
